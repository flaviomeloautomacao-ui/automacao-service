1. Introdução

{% if llm_sections.introducao %}
{{ llm_sections.introducao | safe }}
{% else %}

Este laudo apresenta a análise de riscos de máquinas e equipamentos da empresa {{ metadata.razao_social | default("—") }}, unidade {{ metadata.site | default("—") }}, realizada conforme os requisitos da Norma Regulamentadora NR-12 e normas técnicas aplicáveis (ABNT NBR ISO 12100, entre outras).

O objetivo é identificar perigos, avaliar os riscos associados e propor medidas de proteção que garantam a segurança dos trabalhadores durante operação, manutenção e limpeza dos equipamentos inventariados.

{% endif %}
{# ==================================================================== 2. METODOLOGIA ==================================================================== #}

2. Metodologia

{% if llm_sections.metodologia %}
{{ llm_sections.metodologia | safe }}
{% else %}

A avaliação seguiu as etapas abaixo:

  1. Inventário de máquinas e equipamentos no local avaliado.
  2. Identificação de perigos conforme ABNT NBR ISO 12100:2013.
  3. Classificação do nível de risco (trivial, tolerável, moderado, substancial, intolerável) com base na combinação de probabilidade e severidade.
  4. Hierarquização das ações conforme prioridade (baixa, média, alta, urgente).
  5. Elaboração de recomendações técnicas para eliminação ou redução dos riscos identificados.

Os critérios de probabilidade e severidade estão alinhados aos padrões reconhecidos pela NR-12 e ISO 12100.

{% endif %}
